## Getting Started

To get a local copy up and running follow these simple steps.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Eversmile12/OpenGLGLEW
   ```
2. Build the project using VisualStudio 2019
3. Enjoy OpenGL
